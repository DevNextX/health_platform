# 健康记录平台

> **当前版本**: v1.0.0 | [查看更新日志](CHANGELOG.md)

一个完整的个人健康数据记录与管理平台，提供血压、心率等健康指标的记录、分析和可视化功能。

## 📋 核心功能

### 用户与权限
- **JWT 认证**：注册/登录/登出，邮箱唯一性校验
- **角色管理**：USER / ADMIN / SUPER_ADMIN 三级权限
- **超级管理员**：可提升/降级用户为管理员、重置用户密码

### 健康记录
- 血压（收缩压/舒张压）、心率记录，支持标签和备注
- 数据校验：血压 30-250 mmHg，收缩压 > 舒张压
- 分页、按日期/标签筛选、CSV 导出
- 家庭成员管理：为不同成员录入健康数据

### 数据可视化
- 趋势图表（统一 Y 轴，周/月/全部）
- 异常高亮（血压超 120/80 mmHg 红色标记）
- 图表下载 PNG

## 🚀 快速开始

> 详细说明见 [开发指南](docs/DEVELOPMENT.md)

### 后端 (终端 1)
```cmd
python -m venv .venv && .\.venv\Scripts\activate.bat
pip install -r requirements.txt
set PYTHONPATH=. && python -m flask --app src.app run --port=5000
```

### 前端 (终端 2)
```cmd
cd frontend && npm install && npm start
```

访问 http://localhost:3000
